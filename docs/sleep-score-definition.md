# 睡眠スコアの定義

## 概要

睡眠スコアは、**0-100の範囲**で、あなたの睡眠の質を表す指標です。

## 計算方法

睡眠スコアは以下の4つの要素から計算されます：

### 1. 総睡眠時間スコア（0-40点）

**8時間で満点、6時間以下で0点**

- **8時間以上**: 40点（満点）
- **7-8時間**: 20-40点（線形補間）
- **6-7時間**: 0-20点（線形補間）
- **6時間未満**: 0点

### 2. 深い睡眠の絶対時間スコア（0-35点）

**深い睡眠の絶対時間を評価**（割合ではなく）

- **90分以上**: 35点（満点）
- **60-90分**: 20-35点（線形補間）
- **30-60分**: 0-20点（線形補間）
- **30分未満**: 0点

### 3. REM睡眠スコア（0-15点）

**REM睡眠の絶対時間を評価**

- **120分以上**: 15点（満点）
- **90-120分**: 10-15点（線形補間）
- **60-90分**: 0-10点（線形補間）
- **60分未満**: 0点

### 4. 睡眠の連続性スコア（0-10点）

**深い睡眠の割合が適切かどうかを評価**

- **深い睡眠の割合が15-25%**: 10点（満点）
- **深い睡眠の割合が10-15%または25-30%**: 7点
- **深い睡眠の割合が5-10%または30-35%**: 5点
- **その他**: 2点

## 最終スコア

```
睡眠スコア = 総睡眠時間スコア + 深い睡眠スコア + REM睡眠スコア + 連続性スコア
```

**範囲**: 0-100点
- **0-40点**: 睡眠の質が低い（改善が必要）
- **41-60点**: 睡眠の質が普通（改善の余地あり）
- **61-80点**: 睡眠の質が良好
- **81-100点**: 睡眠の質が最良

## リカバリースコアとの違い

- **リカバリースコア**: HRV、睡眠、心拍数を総合的に評価（回復状態）
- **睡眠スコア**: 睡眠の質のみに焦点を当てた評価

## 注意点

1. **深い睡眠の絶対時間を重視**
   - 割合ではなく、絶対時間（分）で評価
   - 90分以上の深い睡眠が理想

2. **データが不足している場合**
   - 必要なデータ（睡眠時間、深い睡眠、REM睡眠）がない場合は、スコアを計算できません（None）

3. **個人差**
   - 最適な睡眠時間や深い睡眠の時間は個人差が大きい
   - スコアは一般的な基準に基づいているが、個人の体感と合わせて判断することが重要
